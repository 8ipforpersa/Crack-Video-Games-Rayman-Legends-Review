This project is the original version of a digital VFO with analog dial display by Tj Lab(JF3HZB). An Mods  is made by me IU4ALH .

VFOsys       : is fixed step tuning ( normal encoder )
VFOsys_astep : is with adaptive step control ( incremental encoder )
